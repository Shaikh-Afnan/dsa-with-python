DSA with Python

This repository contains a growing collection of Data Structures and Algorithms (DSA) implemented in Python.

The purpose of this repository is to:

- Strengthen problem-solving skills
- Practice core DSA concepts
- Prepare for technical interviews
- Maintain clean, readable, and well-documented code

The repository is updated gradually, as concepts are learned and practiced.

Topics Covered (So Far)

Data Structures:

- Linked List
- Stack (upcoming)
- Queue (upcoming)
- Trees (upcoming)

Algorithms:

- Sorting Algorithms (upcoming)
- Searching Algorithms (upcoming)

How to Run the Code

1. Clone the repository
   git clone https://github.com/Shaikh-Afnan/dsa-with-python.git
2. Navigate to the project folder
   cd dsa-with-python
3. Run any Python file
   python data_structures/linked_list.py
